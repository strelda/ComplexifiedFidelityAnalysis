# ComplexifiedZeroesNotebook
Mathematica code for the article Governing Structure of Survival-Amplitude Zeros as precursor to Dynamical Criticality: [ArXiV](...).

# Files
- `ComplexifiedZeroes.nb`: Mathematica notebook containing the code for Lipkin model.
- `SpinOperatorLibrary.m`: Mathematica package for Hamiltonian creation
- `Lipkin model/`: Directory containing Julia project for fast iterative search for zeroes of Loschmidt amplitude locations.
